    Глобальный переезд    Опубликовано: 26 Mar 2013 22:35

    Сейчас модно отказываться от сервисов гугла. Сейчас модно иметь свой блог как набор html-страниц, которые отдаются со скоростью света и не подвержены всяческим атакам зловредных людей. Так и я, следуя моде, решился на переезд своего бложика с использованием Nikola. Выбор непростой и возможно я ещё передумаю - альтернатив масса, но этот вариант выглядит наиболее функциональным (спасибо @iiiii!)

    В ближайших планах:

    1. Перегнать сюда посты из прежнего блога
    2. Поработать с дизайном
    3. Понять и простить этот инструмент и разобраться как оно работает

    Надеюсь, что результатом работы станет мануал по безболезненному переходу с blogger.com на Nikola, которого сейчас мне чертовски не хватает. Ну что - с почином!

    Интеграция PhpList и Yii    Опубликовано: 27 Aug 2012 12:50

    Условия задачи

    1. Есть стабильная система для рассылки информации разного рода по емаил (например, емаил маркетинг) - PhpList
    2. Есть сайт на фреймворке Yii
    3. Таблица пользователей подсистемы рассылки и сайта - разные.

    Требуется

    Сделать прозрачную интеграцию на уровне: - в административном интерфейсе phpList администратор должен уметь добавлять пользователей из базы Yii; - также необходим компонент для Yii, способный добавлять подписчиков через API; - необходима консольная команда Yii для периодической рассылки по крону.

    Решение

    компонент для фреймворка Yii - https://github.com/mihan007/Yii-phpList
